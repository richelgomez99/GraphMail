### Q: "How do you prevent hallucination?"
**A:** "Agent 3 is a Verification Agent. For each fact extracted by Agent 2, Agent 3 retrieves the claimed evidence emails and uses LLM to verify: 'Does this evidence ACTUALLY support this claim?' ### Q: "Why Graph-First?"
**A:** "Sequential processing handles temporal ambiguity. Early emails are tentative, later emails confirm. Graph-First updates confidence as evidence accumulates, not batch decision-making. Plus, graphs enable complex queries: 'Show all Financial projects with API challenges.'"

**A:** "Agent 1 is deterministic, handles any volume. Agent 2 processes projects in parallel. Agent 3 verifies per-fact with caching. We just processed 320 emails in 15 minutes. With proper batching, scales linearly."
